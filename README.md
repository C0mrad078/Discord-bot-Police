
# Hype Police Discord Bot (Reescrito v6)

## Instalação
1. `pip install -r requirements.txt`
2. Copie `.env.example` para `.env` e coloque `DISCORD_TOKEN`
3. Preencha `config.json` com os IDs do seu servidor/canais
4. `python main.py`

## Setup (no Discord)
- `/setup_prisao`
- `/setup_tickets`

## Notas
- Se a categoria de tickets tem permissões manuais, garanta que o bot/cargo do bot tenha:
  - Ver Canal (View Channel)
  - Gerenciar Canais (Manage Channels)
  - Enviar Mensagens
